# python-course-project-BorisK
Project name: TabTerm

This is a terminal app, for creating, browsing and sharing guitar tabs.
No browser requiered! You can open it on any machine, display server or not, it'll work.
Designed with minimal clutter and maximum music in mind, this app allows you to focus on
playing your favorite song with zero distractions. !!!AND NO ADS!!!
